OWASP

OWASP - Open Web Application Security Project (OWASP) er en ideell organisasjon som fokuserer på å forbedre sikkerheten til webapplikasjoner. OWASP er kjent for sin liste over de ti mest kritiske sikkerhetsrisikoene for webapplikasjoner, kjent som OWASP Top Ten. Denne listen gir utviklere og sikkerhetsprofesjonelle innsikt i de vanligste sårbarhetene som kan utnyttes av angripere, og gir retningslinjer for hvordan man kan beskytte applikasjoner mot disse truslene.

Mitre - ATT&CK er en kunnskapsbase som inneholder informasjon om ulike teknikker og taktikker som angripere bruker for å kompromittere systemer og nettverk. Det er et verktøy som brukes av sikkerhetsforskere, analytikere og forsvarere for å forstå trusler og utvikle strategier for å beskytte seg mot dem. Mitre - ATT&CK gir en strukturert måte å kategorisere og beskrive angrepsteknikker, og det kan hjelpe organisasjoner med å identifisere sårbarheter i sine systemer og implementere effektive sikkerhetstiltak.
